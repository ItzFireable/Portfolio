# Portfolio
This is the source code for the portfolio shown on the fireable.me website. It has been rewritten using Vite + Vue for frontend and ElysiaJS for the backend, both using TypeScript.
To use the project, you need to clone the repository with submodules and initialize them.

### Backend
You need an environment file to run the backend, as it depends on Spotify, Discord, Kamaitachi and Last.fm integration.

Run the commands to launch the backend
```
cd backend
bun run ./src/index.ts
```

### Frontend
Run the commands to launch the frontend
```
cd frontend
bun run dev
```
